# Solo tactics (no second GitHub account)

Goal: unlock earnable achievements without a second personal account.

## Already farmable alone
| Badge | Solo path |
|:--|:--|
| Quickdraw | Open + close issue/PR within 5 minutes |
| YOLO | Merge your own PR with no review |
| Pull Shark | Merge PRs (2 / 16 / 128 / 1024) |
| Public Sponsor | GitHub Sponsors donation (paid) |
| Starstruck | 16+ stars on one public repo (organic) |

## Needs other people (not a second account of yours)
| Badge | Path |
|:--|:--|
| Pair Extraordinaire | Co-authored-by with **any** other real GitHub user who consents (friend, OSS partner, collab issue) |
| Galaxy Brain | 2 answers marked **Accepted** in repo Discussions (answer real questions elsewhere) |

## Do not
- Create sockpuppet accounts only for badges (ToS risk)
- Buy stars / star-exchange farms
- Spam low-quality PRs to strangers

## Order of attack
1. Quickdraw + YOLO + Pull Shark bronze (done in this lab)
2. Keep Pull Shark progress toward 128
3. Galaxy Brain via helpful answers in real Discussions
4. Starstruck via one polished public project + sharing
5. Pair when a real collaborator appears (or mutual OSS pairing with consent)
