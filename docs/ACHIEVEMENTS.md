# GitHub achievements (practice map)

| Achievement | How |
|:--|:--|
| **Quickdraw** | Open + close an issue/PR within 5 minutes |
| **YOLO** | Merge a PR without review |
| **Pull Shark** | Merged PRs: 2 → 16 → 128 → 1024 |
| **Pair Extraordinaire** | Commits with `Co-authored-by:` another GitHub user |
| **Galaxy Brain** | Accepted answers in Discussions |
| **Starstruck** | Stars on your public repos (16+) |

This repo is a sandbox for **real** small docs improvements while practicing flow.
