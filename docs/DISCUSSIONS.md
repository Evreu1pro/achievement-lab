# Discussions

This repo has **Discussions** enabled for Q&A practice and real help.

## Categories (typical)
- **Q&A** — questions that can be marked answered (Galaxy Brain path for answerers)
- **Ideas** — suggestions
- **General** — chat

## For visitors
Ask a clear question: what you tried, what failed, links.

## For maintainers
Mark the best reply as **Answer** when it solves the issue.
