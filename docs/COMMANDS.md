# Commands cheatsheet

## Git
```
git switch -c docs/my-change
git add .
git commit -m "docs: describe change"
git push -u origin HEAD
```

## GitHub CLI
```
gh pr create --fill
gh pr merge --merge --admin
gh issue create --title "..." --body "..."
gh issue close <n>
gh api user
```

## Auth
```
gh auth login
gh auth status
```
